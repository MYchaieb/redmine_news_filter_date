= redmine_news_filter_date
This plugin filter news : it do not show news created on a date before a date you configure on this plugin setting. 

redmine version = 3.1.2
== Install
* #  cd redmine/plugins

* #  git clone https://github.com/MYchaieb/redmine_news_filter_date.git

* #  rake redmine:plugins:migrate RAILS_ENV=production
